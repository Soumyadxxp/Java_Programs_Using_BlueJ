# Java Programs Using BlueJ

## Overview

This repository contains a collection of Java programs developed and practiced using the **BlueJ IDE**. The programs cover fundamental Java programming concepts, object-oriented programming, string manipulation, arrays, recursion, number-based problems, pattern programs, and various programming exercises.

The repository is intended to document my Java programming practice and demonstrate the implementation of core programming concepts through practical programs.

## Development Environment

The programs were developed using **BlueJ**, an integrated development environment designed for learning and working with Java.

### Requirements

To execute the programs, you need:

- Java Development Kit (JDK)
- BlueJ IDE

---

## How to Run the Programs

1. Install the **Java Development Kit (JDK)**.
2. Install **BlueJ**.
3. Open BlueJ.
4. Create or open a Java project.
5. Copy the required Java program from the corresponding document.
6. Compile the Java classes.
7. Create objects or execute the main class as required.
8. Provide input when prompted and observe the output.
